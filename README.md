# SmartSpam: ML-Powered Email Spam Detector

## Introduction
A machine learning-based spam email detection system using scikit-learn algorithms.

## Features
- Detects various spam emails using ML.
- Easy installation and setup.

## Installation
1. Clone the repository:

```shell
   git clone https://github.com/UMARxAI/SmartSpam-ML-Spam-Detector.git
```

2. Install the required Python libraries using pip:
```shell
   pip install scikit-learn==0.22.1
```
3. Run The code 
```shell
   python spam.py
```
